31 March 2025

I chose the Ehomeaffairs website because it is one of the most difficult websites to navigate.

Pros:

  • It is a very simple website to navigate and the information is easy to understand.
  • It is also intuitive to navigate. It feels familiar because the navigation bar is at the top which creates a hierarchy. Also the navigation bar is in a different colour.

    • Cons

  • The user controls are difficult to navigate. The website often does not communicate with the user appropriately. When you have to enter the OTP it consistently does not work.
  • The website is not always operable at times when you request OTPs it does not work.

    • I chose the AFA website because I like the use of colour throughout the website.

    Pros:

  • The website has a video on the homepage which alerts the user what its function is and what it is about.
  • The typography is readable and legible. The fonts on the website are easy to read and understand.

    • Cons

  • The website is not clear. There is not a section on the homepage that alerts the reader what AFA means and what it stands for.
  • The website is very busy and there is a lot of emphasis on animations. The user is not aware where they should focus because their animations are running at the same time.
